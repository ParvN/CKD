
In this project,Chronic Kidney Disease dataset (CKD dataset)  in UCI Machine learning repository UCI Machine Learning Repository [http://archive.ics.uci.edu/ml]  have been used, which includes health parameters of 400 patients with 24 attributes ( excluding the target label class ).Using this dataset and supervised learning algorithms, we can classify whether a new patient has chronic kidney disease or not.

Software Requirements

python 2.7
numpy 
pandas 
sklearn 
matplotlib 
seaborn
